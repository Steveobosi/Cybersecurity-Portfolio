# Junior SOC Analyst Hands-On Labs (LetsDefend)

Completed hands-on SOC labs in a simulated enterprise environment, focusing on alert triage, incident investigation, and threat analysis.

**Tasks Completed:**
- Investigated SIEM alerts and monitored security events  
- Analyzed phishing emails, malicious URLs, domains, and IP addresses  
- Conducted static malware analysis on malicious documents using REMnux  
- Analyzed VBA macros, execution triggers, and file metadata  
- Extracted and documented Indicators of Compromise (IOCs)  
- Validated findings using VirusTotal and Hybrid Analysis  
- Investigated web attacks: SQL Injection (SQLi), Cross-Site Scripting (XSS), Local & Remote File Inclusion (LFI/RFI), Directory Traversal, Open Redirection, Brute-force and authentication attacks  
- Analyzed web server logs and HTTP requests to identify attack patterns  
- Distinguished true positives from false positives during alert triage  
- Followed Tier-1 SOC workflows: triage, validation, enrichment, escalation  
- Documented investigations following SOC reporting best practices  

**Tools Used:** Splunk SIEM, REMnux, VirusTotal, Hybrid Analysis, oletools, exiftool  
**Skills Gained:** Endpoint Security, Log Analysis, Incident Response, SOC Operations
