# LetsDefend SOC Lab

Performed hands-on SOC investigations using simulated SIEM alerts, focusing on log monitoring, alert triage, and incident response.

**Tasks Completed:**
- Monitored and analyzed security events and logs in a SOC environment  
- Investigated phishing emails, malicious URLs, domains, and IP addresses  
- Conducted static malware analysis on malicious documents using REMnux  
- Analyzed web-based attacks: SQL Injection (SQLi), Cross-Site Scripting (XSS), Local & Remote File Inclusion (LFI/RFI), Directory Traversal, Open Redirection, Brute-force and authentication attacks  
- Extracted and documented Indicators of Compromise (IOCs)  
- Validated findings using VirusTotal and Hybrid Analysis  
- Followed Tier-1 SOC workflows: alert triage, validation, enrichment, escalation  
- Documented investigations following SOC reporting best practices  

**Tools Used:** Splunk (SIEM), REMnux, VirusTotal, Hybrid Analysis, oletools, exiftool  
**Skills Gained:** Playbook Creation, Incident Response, Endpoint Security, Log Analysis
