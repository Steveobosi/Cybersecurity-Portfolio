# Self-Directed SOC Analyst Home Lab

Built and operated a personal home lab environment to simulate Tier-1 SOC operations, focusing on log monitoring, detection engineering, and incident triage using real tools.

**Tasks Completed:**
- Monitored logs and detected suspicious activity  
- Created custom alerts and detection rules  
- Performed incident triage and validated findings  
- Simulated SOC workflows using real-world tools  

**Tools Used:** Splunk, REMnux, VirusTotal, Hybrid Analysis  
**Skills Gained:** Splunk, Security Monitoring, Incident Response, Detection Engineering
